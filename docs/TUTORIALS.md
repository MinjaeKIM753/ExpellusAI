## Core Concepts

- **Natural Language Commands** — Type what you want in plain English. ExpellusAI reads your screen and executes the task.
- **Visual Flow Canvas** — Build multi-step automations visually. Drag, drop, connect, and run.
- **Pause & Resume** — Stay in control. Pause any automation mid-execution, review what's happening, and resume or adjust.
- **AI Browser** — A dedicated browser profile for web automation that keeps your personal browsing separate.

## Use Cases

### Desktop Automation
> *"Open Notepad, write a to-do list for today, and save it to my Desktop"*

ExpellusAI can interact with any Windows application — office suites, file managers, system settings, and more.

### Browser & Research
> *"Search Google for the latest AI news and summarize the top 3 results"*

Automate web browsing, form filling, data extraction, and research tasks.

### System Control
> *"Change my display resolution to 1920x1080 and enable dark mode"*

Modify system settings, manage files, and control your PC environment.

### Development
> *"Open VS Code, create a new Python file, and write a hello world program"*

Automate development workflows, terminal commands, and code editing tasks.
